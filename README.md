# OpenClaw Tools 🦞

A personal collection of [OpenClaw](https://github.com/openclaw/openclaw) tools, skills, and configurations.

OpenClaw is a personal AI assistant that runs on your own devices and connects to your favorite messaging platforms.

## 📋 What's Inside

- **`skills/`** - Custom OpenClaw skills and tools
- **`configs/`** - Example configuration files for OpenClaw setup
- **`docs/`** - Documentation and guides

## 🚀 Getting Started

### Prerequisites

- Node.js >= 22.0.0
- OpenClaw installed globally: `npm install -g openclaw@latest`

### Installation

1. Clone this repository:
```bash
git clone https://github.com/npeprah/openclaw.git
cd openclaw
```

2. Install dependencies:
```bash
npm install
```

3. Configure OpenClaw:
   - Copy example configs from `configs/` directory
   - Update with your actual values
   - Follow the [OpenClaw documentation](https://docs.openclaw.ai)

### Using Custom Skills

To use the skills in this repository with OpenClaw:

1. Set your workspace skills directory to include this repo's `skills/` folder
2. Or symlink individual skills to your OpenClaw skills directory
3. Restart your OpenClaw gateway

See the [OpenClaw Skills Documentation](https://docs.openclaw.ai/tools/skills) for more details.
