# Configuration Examples

This directory contains example configuration files for OpenClaw.

## Files

- `example.config.json` - Example OpenClaw configuration
- `channels.example.json` - Example channel configuration

## Usage

1. Copy the example files and remove the `.example` suffix
2. Update the configuration with your actual values
3. Reference these in your OpenClaw setup

For more information, see: https://docs.openclaw.ai/gateway/configuration
