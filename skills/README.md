# Example OpenClaw Skill

This is an example skill that demonstrates how to create a custom skill for OpenClaw.

## Structure

Skills should export:
- `name`: The skill name
- `description`: What the skill does
- `execute`: The function that runs when the skill is invoked

## Usage

Skills can be loaded into OpenClaw by placing them in your skills directory and configuring them in your OpenClaw workspace.

See the OpenClaw documentation for more details: https://docs.openclaw.ai/tools/skills
